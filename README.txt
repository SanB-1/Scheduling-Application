                   ****    SCHEDULER    ****

This is a scheduling application that allows users to schedule appointments with different customers.

Application Version:      1.0
Current Date:             5/20/2023
IDE:                      Intellij IDEA 2021.1.3 (Community Edition)
JDK Version:              17.0.1
JavaFX Version:           11.0.2
MySQL Connector Version:  8.0.25

How to run: 
                   * Include all the required libraries (JavaFX, MySQL connector).
                   * Create the MySQL database using the provided script "database.txt" and the following information:
                     Username: sqlUser Port: 3306 Password: Passw0rd! Database Name: client_schedule
                   * Compile and Execute the Main.java file present in the "Model" folder.
